# The Technical Team 10

## Requirements
Node v12.18.3 or higher
```
https://nodejs.org/en/download/
```

## Front End Installation
```
npm install
```

## Run the React Application
```
npm start
```
This will start the front end and load it on your preferred browser
link: `localhost:3000`
